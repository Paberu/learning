# Изучить, как работать с файлами markdown.

Мне повезло, что свою базу знаний я веду в приложении Obsidian, хранилище которого представляет собой семейство папок с md-файлами внутри. Соответственно, если посмотреть на GitHub мои отчёты по предыдущим курсам, можно заметить, что многие из них сохранены именно в формате .md просто потому, что они отображаются самим GitHub значительно лучше, чем голый txt или не обременённый css-файлами html.
